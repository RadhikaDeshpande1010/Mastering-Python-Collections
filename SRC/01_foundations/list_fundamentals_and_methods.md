### Q1. Check if a number exists in a list.

**Input:** 
`list1 = [10, 20, 30, 40, 50]`
`number = 10`

**Output:** `true`

```python
# check_number_in_list.py

list1 = [10, 20, 30, 40, 50]
number = 10
res1 = number in list1
print(res1)

Output:
True
```
